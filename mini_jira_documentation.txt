## âœ¨ Core Features (5-Day Scope)

### Day 1-2: Foundation & Authentication
#### 1. User Authentication & Authorization
- User registration with email and password
- User login with JWT token generation
- Basic user profile (name, email, role)
- Logout functionality
- Password validation

#### 2. Project Management
- Create new projects
- View list of all projects
- Basic project details (name, description, created date)
- Delete projects

### Day 3: Task Management Core
#### 3. Task/Issue Creation & Management
- Create tasks with:
  - Title
  - Description
  - Priority (Low, Medium, High)
  - Status (To Do, In Progress, Done)
  - Assignee (team member)
  - Due date
- View task details
- Edit task information
- Delete tasks

#### 4. Task Status Management
- Update task status via drag-and-drop or dropdown
- Three basic statuses: **To Do**, **In Progress**, **Done**

### Day 4: Board View & Assignment
#### 5. Kanban Board View
- Visual board with three columns (To Do, In Progress, Done)
- Drag and drop tasks between columns
- Task cards showing key information

#### 6. Task Assignment
- Assign tasks to team members
- View tasks assigned to specific users
- Filter tasks by assignee

### Day 5: Dashboard & Polish
#### 7. Dashboard
- Overview of all projects
- Task statistics (total, completed, pending)
- Recent tasks
- Tasks assigned to logged-in user

#### 8. Basic UI/UX
- Responsive design for mobile and desktop
- Clean and intuitive interface
- Loading states and error handling
- Basic notifications for actions
